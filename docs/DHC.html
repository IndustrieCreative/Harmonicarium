Description

The Dynamic Harmonics Calculator class

This is the computational kernel for the frequency/midicent tables.
Manage an route the communications from and to the other App components.

Parameters
Name Type Description
id string

The DHC id, in format harmonicariumID-dhcID (eg. '1-0')

The HARMONICARIUM instance to which the this DHC must refer
